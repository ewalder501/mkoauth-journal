# mkoauth-journal

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅用于存放各类 HTML 文档，以便于版本管理和在线访问。

## 目录说明

```
mkoauth-journal/
├── index.html          # 仓库首页，包含页面索引
├── pages/              # 存放独立 HTML 页面的目录
│   └── ...             # 各归档页面文件
└── README.md           # 本文件
```

- `index.html`：仓库的入口页面，提供内部页面的导航与列表。
- `pages/`：所有独立 HTML 页面均存放于此目录下，文件名应反映页面内容或用途。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立文件，不依赖外部资源（除 CDN 或公共库外）。归档内容涵盖各类静态文档、工具页面或实验性页面，适合直接通过 GitHub Pages 或其他静态托管服务发布。

如需添加新页面，请遵循以下规范：

1. 页面文件放置于 `pages/` 目录下。
2. 文件名使用小写英文、数字及连字符，避免空格和特殊字符。
3. 页面顶部建议包含简短的注释说明（如标题、创建日期）。
4. 避免在页面中嵌入敏感信息或外部跟踪代码。

## 维护说明
